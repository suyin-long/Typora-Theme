## 简介

- 基于 [Drake V2.9.2](https://theme.typora.io/theme/Drake/) 修改的 Typora 主题

## 使用方法

- 下载 `drake` 文件夹和想用的 `css` 样式文件
- 通过 `偏好设置 -> 外观 -> 打开主题文件夹` 打开 theme 目录
- 将下载好的文件和文件夹复制到 `theme` 目录下，然后重启 Typora
- 然后在 `主题` 菜单栏下选择刚才添加的主题
